### How To

First Clone Application

```
git clone https://github.com/uchup07/hacktiv8-nodejs-testing.git
```

#### 1. Install NPM Package

```cmd
npm install
```

#### 2. Create environment

rename file ``.env.example`` to ``.env``

```cmd
mv .env.example .env
```

#### 3. RUNNING THE APPS

```cmd
npm run start
```

#### 4. Test Apps

```cmd
npm run test
```
